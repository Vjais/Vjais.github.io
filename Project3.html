PROJECT VISION

This project is an immersive Virtual Reality game developed for CS491.The idea behind this project was to get experience creating virtual worlds for people to move around in and interact.The user should be able to move through a series of outdoor scenes,either by walking or teleporting in given space.Each scene is designed in a unique way that user experiences immersive effect.

PROJECT THEME and DESCRIPTION

It is a single player outdoor adventure VR game.The player has to move into different levels and find the portals to transition into the next level.In this game,we have created 5 levels.Each levels is unique.It starts from the forest.As the player moves into the next level,the environment becomes darker.There are certainm tools to cross the barriers,making use of objects around and escaping from certain creatures..The main theme shows that the worls is peaceful and calm at the initial stage.In last stage,the evil character shows up and with a snap half of the world is destroyed.The playerâ€™s movement will involve walking and teleporting.For more immersion we have added haptic effects.All the scenes have been mentioned below.The scenes are inspired from pop culture so that player can relate with every scenes.All the scenes are desdigned from scratch.Please check the video below.

SCENE 1:FOREST AND UNDERWATER SCENE

This scene is divided into two parts.The first part is where you can explore forest built on terrain.There are certain animals in the forest like birds,rat,snake,bear,monkeys and cub.There is an axe at the the treehouse.The signboards to direct.The next scene starts from underwater.There are different fishes,octopus,boat,starfish,shells,plants.There is a chestbox which has gem.The player has to pickup the gem and add it to rim lying next to box.There will be portal appearing which is key to next level.

SCENE 2:UPSIDE DOWN SCENE

This scene is inspired from stranger things.In this level you can kill demogorgans.There will be many mysterious creatures in this scene such as skeleton,zoombies.There are few animals spider,lion,elephants etc.At the end there will be a portal to move into next. level.

SCENE 3:DRAGON WORLD SCENE

In this scene,there are dragons which you have to kill to move to next level.The key here is egg which will be present in nest.You will only find it if you kill all the dragons using bow and arrow.

SCENE 4:CAVE SCENE

The cave scene is inspired from egyptian theme.You will find mummy and sculpture.There is a light saber game added in this level.
