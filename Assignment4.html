Initial thought on applications :

This week,I got a chance to explore the sky using some amazing applications.As complex the word astronomy sounds it's more hard to imagine the celestial bodies.The invent of modern technologies has given rise to certain application which has given a direction to these unimaginable things.Some of few application are available on smartphones that provides window to universe.

For this assignment,I tried three different applications.The first application that I tried was Skyview.This application gives a real time 360 degree magnified view of Planets, stars, constellations, and satellites on your phone screen using sensors.It is quite an engaging application.Asthetically it is very pleasing and soothing for eyes.The sound feature intensifies the experience.On tapping the stars or planet,certain information(fun fact) pops up on screen.The features that impressed me the most was the in-build compass that also reminds of the direction the user is moving and time travel so thst user can track activities that happened in the past.The only part which is bit odd is absence of manual mode to move around objects.
