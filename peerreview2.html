Choice 2:AR Runner

For this week,we have to review two student's choice that were presented during week 12 and 13.My second choice for the review is AR Runner presented by Sai Phaltankar.Augmented reality has been changing the phase of android application by blending the digital and real worlds by using your camera lens.The user can find these applications intriguing as it allows user to interact in real time.The embedding of augmented reality with android and ios applications have given rise to some very cool applications.As AR applications are evolving,one of such interesting application is AR runnner.It is interesting in a way that it encourages player to workout and is also a good source of entertainment.

AR runner is a augmented reality mobile game which involves the user to move around in the surrounding.It is a free application available on iOS App Store.All you need to do is install the application on your device.The application initially scans the area select by the player and then set the track with checkpoints in the application.The image below shows the track built after scan.The player has to quickly cover these checkpoints.There are two editions of this game i.e a free edition and a premium edition.It can be played in both the modes as single or multiplayer.The free version has three different versions.The first version is Normal mode which has the simply has the track.The second mode is 30 Second Rush,the player has to cover certain checkpoints within 30 seconds.The third version is Enemies,the player has to avoid enemies hitting the checkpoints.The premium version of the game offers two new field shapes and two additional modes.There is also statistics available which compares the progress with the other players through the leaderboard.There are also in built challanges that player can select accordingly.
