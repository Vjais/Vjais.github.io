Description

For project-3,we are planning to create a multiplayer adventure VR game.To make this game more interactive,we have decided to make it a multi-level game where player has to survive through all the levels to reach the final level.In this game,we want to implement multiple props that will make this game more challenging and fun. In every level,the player will be facing a different environment and difficulty increases with each passing level.As per a rough idea,the initial stage will involve finding basic tools,making use of objects around and escaping from certain creatures.In next stage,the game takes a different world where player has to find clues,solve the mystery and also survive through several barriers.In the final stage,player has to fight with game character in order to win the game.The playerâ€™s movement will involve walking,teleporting and to move into different levels.The game terminates if the player fails to cross any of these level.The clues could be in form of sound,animation,text etc. This game is inspired currently existing game Survival Evolved and Goblet of fire .In these games,the player uses some of the strategies discussed above.
