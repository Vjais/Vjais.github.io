TOOLS USED : Unity 3D version 2018.2

PLATFORM: Vuforia

Unity 3D is a cross-platform game engine developed by Unity Technologies that supports AR and VR development.It is used to create 2D and 3D games as well as simulations.Currently,the primary language used for this engine is C#.All the initial installation and configuration was straightforward.One striking thing about unity is availability of enough tools to support several platforms for various integration in application.The high resolution graphics and the responsive interaction makes it more popular.The only disadvantage is that it consumes a lot of memory .

Unity integrates the Vuforia Engine, making it even easier to create augmented reality experiences for both handheld and headworn devices.The integration provides flexibility for developers.The available documentation makes it easy to understand for beginners.It's application can help in better visualization and modelling.

Above is a screenshot of example utilizing Vuforia,an astronaut mascot standing on the Astronaut image target.The one of the best part is user can import their target image and add 3D objects of own choice by enhancing it.All in all,it's a great software and has a promising future for augmented reality.
